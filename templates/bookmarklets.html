PPMグラフ ラベル追加
PPMグラフの散布図に、選択したセル範囲のデータラベルを追加するExcel VBAマクロ（PPM_add_label.bas）です。
インポート手順
  1. Excelを開き、Alt + F11 でVBAエディターを起動
  2. メニューの 「ファイル」→「ファイルのインポート」 を選択
  3. ダウンロードした PPM_add_label.bas を選択して開く
  4. VBAエディターを閉じる（Alt + F4）
クイックアクセスツールバーへの追加
  1. Excelの 「ファイル」→「オプション」→「クイックアクセスツールバー」 を開く
  2. 「コマンドの選択」で 「マクロ」 を選ぶ
  3. 一覧から 「PPMグラフにデータラベル範囲選択し設定」 を選び、「追加」 をクリック
  4. 「OK」 で保存
使い方
  1. PPMグラフ（散布図）をクリックして選択状態にする
  2. ツールバーのボタン（またはマクロ実行）でマクロを起動
  3. ダイアログが表示されたら、ラベルにしたいセル範囲を選択して 「OK」
マクロを有効にする（トラストセンター）

マクロがブロックされる場合は、以下の手順でExcelのトラストセンターでマクロを有効にしてください。

  1. Excelの 「ファイル」→「オプション」 を開く
  2. 「トラストセンター」→「トラストセンターの設定」 をクリック
  3. 左メニューの 「マクロの設定」 を選択
  4. 「VBAマクロを有効にする（推奨しない。危険なコードが実行される可能性があります）」 を選び、「OK」 で保存

信頼できるファイルのみマクロを有効にしてください。使用後は設定を元に戻すことを推奨します。
