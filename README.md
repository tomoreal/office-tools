# 財務データ変換ツールセット (XBRL & CSV)

このリポジトリには、EDINETのXBRLファイルおよびEOLのCSVファイルを、分析しやすいExcel形式（横展開）に変換するためのツールセットが含まれています。
edinetにapiでアクセスして、XBRLファイルをダウンロードし、excelに変換するツールを開発します。2026/3/21から開始します。

## 1. ツール概要

### ① XBRL変換ツール (メインプログラム：`convert_xbrl_to_excel.py`)
- **用途:** EDINETからダウンロードしたXBRL (zip) ファイルを解析し、財務諸表をExcel形式に変換します。
- **特徴:** 
  - 連結・単体の自動判別
  - 日本基準・IFRSの両方に対応
  - 複数年・複数企業のデータを整理して出力

### ② CSV変換ツール (メインプログラム：`app2.js`)
- **用途:** EOL (Financial Data DB) からダウンロードしたCSVファイルを、年度が横に並ぶ形式に変換します。
- **特徴:** 
  - 縦形式のCSVを数表形式に再構成
  - スマートソート機能により、年度順に整理

### ③ CSV変換ツール　windowsのVBS用　※サーバーに設置は不要(`財務データ横展開ツール.vbs`)
- **用途:** EOL (Financial Data DB) からダウンロードしたCSVファイルを、年度が横に並ぶ形式に変換します。web版と同様のプログラムです。
- **使い方:** 変換したいCSVファイルを、このVBSスクリプトにドラッグ＆ドロップしてください。
---

## 2. 使い方
WEBアプリの操作方法は、各アプリのトップ画面を参照してください。

### XBRL変換ツールの利用
1. [EDINET](https://disclosure2.edinet-fsa.go.jp/weee0050.aspx) で対象のXBRLファイル (zip) をダウンロードします。
2. 本ツールのWeb画面からzipファイルをアップロードします。
3. 変換されたExcelファイルが自動的にダウンロードされます。

### CSV変換ツールの利用
1. [EOL](https://ssl.eoldb.jp/EolDb/UserLogin.php) にアクセスし、企業を検索します。
2. 「財務データ（有報）」→「原文（財務データCSV）」画面を開きます。
3. 必要な表（連結貸借対照表、損益計算書、CF計算書など）の「通期」にチェックを入れます。
4. **「縦形式」**を選択して、CSVファイルをダウンロードします。
5. 本ツールのWeb画面から、または専用のVBSスクリプトにドラッグ＆ドロップして変換を行います。

## 3. セットアップ・デプロイ

サーバーへの設置方法については [DEPLOY.md](./DEPLOY.md) を参照してください。

### 構成ファイル

#### EDINETのXBRL変換ツール
- `index.cgi`: CGI環境でXBRL変換ツールを動作させるための設定ファイル。
- `app.py`: XBRL変換ツールのファイル受け渡し用プログラム。
- `convert_xbrl_to_excel.py`: XBRL変換エンジンのコア。EDINETの複雑なデータ構造を解析し、Excelを生成するメインロジック。
- `templates/`: XBRL変換ツール/CSV変換ツールのHTMLテンプレートディレクトリ。
    - `index.html`: XBRL変換ツールのメイン画面テンプレート。
    - `bookmarklets.html`: XBRL変換ツール用のブックマークレット（ブラウザのお気に入りに追加して使用する、効率化スクリプト）の管理画面。

#### EOLのCSV変換ツール
- `index.html`: EOLのCSV変換ツールのメインUI画面。
- `style.css`: index.htmlがブラウザで読み込まれる際、デザインやレイアウトを適用。
- `app2.js`: EOLのCSV変換ツールのメインプログラム。
- `templates/`: XBRL変換ツール/CSV変換ツールのHTMLテンプレートディレクトリ。
    - `csv_bookmarklets.html`: CSV変換ツール用のブックマ ークレット（ブラウザのお気に入りに追加して使用する、効率化スクリプト）の管理画面。

#### 共通・Webプラットフォーム関連
- `DEPLOY.md`: サーバー設置・環境構築の手順書。
- `requirements.txt`: XBRL変換ツール実行に必要なPythonライブラリの一覧。

#### CSV変換ツール　windowsのVBS用　※サーバーに設置は不要
- `財務データ横展開ツール.vbs`: 複数のExcelファイルを1つのブックにまとめるための補助ツール。
- `エクセル結合.vbs`: 複数のExcelファイルを1つのブックにまとめるための補助ツール。
